## 📞 GET STARTED NOW

```powershell
# 1. Clone Stealth-Technology
git clone https://github.com/FOX-HEIGHT-BUILD/Stealth-Technology.git
cd Stealth-Technology

# 2. Setup environment (Run as Administrator)
.\config\environment-setup.ps1 -Permanent -Verify

# 3. Build all components
.\build-scripts\build-all.ps1 -Language all

# 4. Test everything
.\build-scripts\test-all.ps1 -TestType all

# 5. Start Local API
cd local-api
python local_api_core.py

# 6. Register functions (in another terminal)
cd examples
python register_all_functions.py

# 7. Open dashboard
Start-Process "file:///C:/Development/Mighty-Power/frontend/index.html"
```
